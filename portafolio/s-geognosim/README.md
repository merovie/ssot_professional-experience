# 🛰️ Macro-Proyecto: GeoGnoSim

## 🎯 Contexto General
Este proyecto nació de la necesidad de acelerar el condicionamiento de datos en software minero para reducir la incertidumbre geológica. Lideré la transformación de un algoritmo lento (TRL 3) a una herramienta comercial de alto desempeño (TRL 8/9). El software original presentaba una limitación crítica: la lentitud en el condicionamiento de datos, lo que impedía su adopción industrial. Mi desafío fue liderar una reingeniería completa, apalancando fondos públicos para mejorar el algoritmo, validar su impacto técnico y diseñar la estrategia de salida al mercado (Go-To-Market) para lograr las primeras ventas.
